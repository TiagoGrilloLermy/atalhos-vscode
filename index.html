Atalhos do Visual Studio Code

Windows

  • Ctrl + P: Abrir arquivo
  • Ctrl + Shift + N: Abrir nova janela
  • Ctrl + S: Salvar arquivo
  • Ctrl + Shift + S: Salvar todos os arquivos
  • Ctrl + Z: Desfazer
  • Ctrl + Y: Refazer
  • Ctrl + C: Copiar
  • Ctrl + X: Recortar
  • Ctrl + V: Colar
  • Ctrl + F: Localizar
  • Ctrl + H: Substituir
  • Ctrl + /: Comentar/Descomentar linha
  • Ctrl + D: Selecionar próxima correspondência
  • Ctrl + U: Desfazer seleção
  • Ctrl + K, Ctrl + S: Mostrar todos os atalhos do teclado
  • Ctrl + Shift + L: Selecionar todas as correspondências
  • Ctrl + F12: Ir para a definição
  • Ctrl + B: Alternar barra lateral
  • Ctrl + J: Alternar painel de saída
  • Ctrl + `: Alternar terminal integrado
  • Ctrl + .: Ações rápidas
  • Alt + Up/Down: Mover linha para cima/baixo
  • Ctrl + Alt + Up/Down: Duplicar linha para cima/baixo
  • Ctrl + Shift + K: Excluir linha
  • Alt + Click: Adicionar cursor
  • Ctrl + Alt + Click: Adicionar cursor acima/abaixo
  • Ctrl + Alt + Shift + Click: Adicionar cursor em todas as correspondências selecionadas
  • Alt + F12: Encontrar todas as referências
  • Ctrl + T: Navegar entre símbolos
  • Ctrl + K, Z: Entrar no modo de tela inteira
